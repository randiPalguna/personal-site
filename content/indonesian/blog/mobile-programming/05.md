## Komponen Button
Untuk dapat memiliki button yang bisa ditekan dan otomatis akan mengganti gambar dice yang pada aplikasi, dibutuhkan komponen `Button` dengan argumen `onClick`. Argumen ini dapat mengubah nilai mata dadu secara random dari 1..6 ketika user menekan button tersebut.
```java
Button(
    onClick = { result = (1..6).random() },
    shape = RoundedCornerShape(20),
) {
    Text(
        text = "Roll"
    )
}
```
<br>

## Variable State yang Mutable
Agar aplikasi dapat mengingat nilai mata dadu yang didapat setelah menekan button, nilai mata dadu tersebut perlu disimpan kedalam suatu variable mutable dan dapat diingat oleh aplikasi dalam suatu state.
```java
var result by remember { mutableIntStateOf(1) }
```
{{< notice "note" >}}
Dari kode diatas, variable result merupakan variable yang kita inginkan, dimana state awal aplikasi menunjukkan mata dadu ke 1. Ini menunjukkan bahwa saat setelah aplikasi baru saja dibuka, gambar dadu yang muncul adalah dadu bermata satu.
{{< /notice >}}

<br>

## Komponen Gambar yang Dapat Berubah Sewaktu Variable State Berubah
Sebelum membangun komponen `Image`, kita perlu untuk mendeklarasikan sebuah variable yang meng-assign ID gambar sesuai dengan nilai mata dadu yang diperoleh dalam suatu state. Variable ini nantinya akan digunakan untuk meng-assign argumen `painter = paninterResource()` pada komponen `Image()`.
```java
val imageResource = when (result) {
    1 -> R.drawable.dice_1
    2 -> R.drawable.dice_2
    3 -> R.drawable.dice_3
    4 -> R.drawable.dice_4
    5 -> R.drawable.dice_5
    else -> R.drawable.dice_6
}
```

```java
Image(
    painter = painterResource(imageResource),
    contentDescription = "1"
)
```

```java
package com.example.diceroller

import androidx.compose.runtime.setValue

class MainActivity : ComponentActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        enableEdgeToEdge()
        setContent {
            DiceRollerTheme {
                Scaffold(modifier = Modifier.fillMaxSize()) { innerPadding ->
                    DiceRollerApp(
                        modifier = Modifier.padding(innerPadding)
                    )
                }
            }
        }
    }
}

@Preview
@Composable
fun DiceRollerApp(modifier: Modifier = Modifier) {
    DiceWithButtonApp(modifier
        .fillMaxSize()
    )
}

@Composable
fun DiceWithButtonApp(modifier: Modifier) {
    var result by remember { mutableIntStateOf(1) }
    val imageResource = when (result) {
        1 -> R.drawable.dice_1
        2 -> R.drawable.dice_2
        3 -> R.drawable.dice_3
        4 -> R.drawable.dice_4
        5 -> R.drawable.dice_5
        else -> R.drawable.dice_6
    }
    Column (
        modifier = modifier,
        verticalArrangement = Arrangement.Center,
        horizontalAlignment = Alignment.CenterHorizontally
    ) {
        Image(
            painter = painterResource(imageResource),
            contentDescription = "1"
        )
        Spacer(modifier = Modifier.height(16.dp))
        Button(
            onClick = { result = (1..6).random() },
            shape = RoundedCornerShape(20),
        ) {
            Text(
                text = "Roll"
            )
        }
    }
}
```
