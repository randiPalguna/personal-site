## Komponen UI
### Komponen Text
"Happy Birthday James!" dan "From Emma" adalah contoh komponen text yang dibangun melalui Jetpack Compose menggunakan metode `Text()`.

```java
Text(
    text = "Happy Birthday $name!",
    fontFamily = FontFamily(Font(R.font.wow_font)),
    color = Color.White,
    fontSize = 64.sp,
    lineHeight = 64.sp,
    textAlign = TextAlign.Center,
    modifier = Modifier.fillMaxWidth()
)
Text(
    text = "From $from",
    fontFamily = FontFamily(Font(R.font.wow_font)),
    color = Color.White,
    fontSize = 32.sp,
    textAlign = TextAlign.End,
    modifier = Modifier
        .fillMaxWidth()
        .padding(end = 20.dp)
)
```

### Komponen Column
Agar tulisan "Happy Birthday James!" berada diatasnya "From Emma", maka metode `Column()` perlu digunakan.
```java
Column (
    verticalArrangement = Arrangement.Center,
    modifier = Modifier.fillMaxSize()
) {
    Text(
      ...
    )
    Text(
      ...
    )
}
```

### Komponen Image
Untuk memunculkan gambar di dalam aplikasi, kita dapat menggunakan metode `Image()`.
```java
Image(
    painter = painterResource(R.drawable.peak_bg),
    contentDescription = null,
    contentScale = ContentScale.Crop,
    modifier = Modifier
        .fillMaxSize()
)
```

### Komponen Box
Untuk mengatur tata letak antara komponen gambar dan komponen lainnya seperti tulisan sedemikian hingga gambar muncul dibawahnya tulisan (agar tidak menutupi tulisan), maka diperlukan metode `Box()`.
```java
Box(
    modifier = modifier.fillMaxSize()
) {
    Image(
      ...
    )
    Column(
      ...
    ) {
        Text()
        Text()
    }
}
```
