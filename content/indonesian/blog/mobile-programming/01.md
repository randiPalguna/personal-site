## Membuat Aplikasi Hello Android
Untuk membuat aplikasi sederhana, seperti aplikasi yang hanya menampilkan tulisan "Hello Android!", kita membutuhkan aplikasi bernama Android Studio. Android Studio adalah IDE untuk mengembangkan sebuah aplikasi Android. Pengembangan aplikasi di Android studio biasanya menggunakan Kotlin sebagai bahasa pemrograman dan Jetpack Compose sebagai pembangun UI. Kita dapat mengunduh Android Studio pada [link](https://developer.android.com/studio) ini. Berikut adalah cara membuat aplikasi yang menampilkan text "Hello Android!".

### Menyambungkan Mobile Phone dengan Android Studio
Setelah menunggu Android Studio mengenerate kode template yang diperlukan. Kita dapat menyambungkan handphone kita ke Android Studio untuk dapat menjalankan aplikasi yang telah kita buat. Adapun cara untuk menyambungkannya adalah dengan menggunakan Wireless Debugging (Wi-Fi) atau menggunakan USB Debugging. Disini saya menggunakan USB Debugging yaitu dengan memasangkan kabel USB dari komputer ke handphone dan mengaktifkan menu "USB Debugging" pada aplikasi pengaturan di handphone. Setelah melakukan hal tersebut, mobile phone akan otomatis tersambung dengan Android Studio dan siap untuk menjalankan aplikasi yang telah kita buat.
